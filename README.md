### 🚀 Selected Projects

#### [Vamooze](https://web-stg.vamooze.com) — Logistics Operations Platform

**Product Architect & Sole Backend Engineer**

Independently designed, built, integrated, and deployed the production backend for an asset-leasing and logistics platform.

- Built the API with **TypeScript, FeathersJS, PostgreSQL, Redis, REST, and WebSockets**.
- Implemented real-time rider matching, delivery acceptance, scheduled deliveries, location updates, tracking links, and lifecycle state transitions.
- Designed asset, lease, maintenance, business, rider, request, wallet, and transaction domains.
- Used **BullMQ + Redis** for dispatch matching, recurring location requests, scheduled deliveries, and subscription reminders.
- Integrated **Paystack** wallet funding, bank validation, withdrawals, transfers, signed webhooks, and duplicate-safe transaction processing.
- Added notifications across **OneSignal, Termii SMS, email, and real-time socket channels**.
- Automated production deployment to **AWS Lightsail** with **GitHub Actions and PM2**.

#### [Trades Trek](https://www.tradestrek.com) — Stock-Trading Simulator

**Product Architect & Sole Backend Engineer**

Independently architected and built the backend for a virtual stock-trading and financial-learning platform.

- Built the platform with **NestJS, TypeScript, MongoDB, Bull, and Redis**.
- Implemented buy, sell, short, and buy-to-cover workflows, holdings, virtual buying power, portfolio valuation, and profit-and-loss calculations.
- Built market-data ingestion, stock analytics, chart data, sectors, gainers/losers, watchlists, order histories, and administrative reporting.
- Developed public and private competitions, invitations, leaderboards, rankings, portfolio resets, and configurable trading rules.
- Delivered learning modules, quizzes, user progress tracking, news, alerts, and push/email notifications.
- Integrated **Paystack** and **Apple App Store** subscriptions with discounting, gifting, expiry handling, duplicate-reference protection, and payment reconciliation.
- Automated staging and production deployments to **Hetzner** using **GitHub Actions and PM2**.

#### [Nigeria HealthLMIS](https://healthlmis.ng) — National Health Supply-Chain Platform

**Software Engineer at Field**

Contributed production features across a multi-service healthcare supply-chain platform used by public-health programs.

- Designed a two-pass, budget-optimised allocation algorithm that distributes constrained budgets across hundreds of products while targeting minimum months of stock.
- Rolled out the allocation workflow across multiple programs behind tenant-scoped feature flags.
- Shipped improvements spanning **orders, shipments, pricing, inventory, reports, and last-mile delivery**.
- Worked across backend services, GraphQL APIs, operational workflows, and frontend integration points in a large production monorepo.
- Delivered **79+ merged pull requests and approximately 26,000 lines of production changes** across the platform.

#### AgroMall — Digital Agriculture Platform

Built offline-capable data workflows, GPS-based land validation, and automated inventory reconciliation for a platform managing **20,000+ farmers**.
